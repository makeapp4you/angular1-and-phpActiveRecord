Tool

Thư viện chứa nhiều chức năng hữu ích cho các bạn sau này khi xây dựng và phát triển Website. Bao gồm những chức năng như chuyển đổi ký tự Việt Nam thành ASCII hay lấy tên file Thumbnail có ký tự đặt trước extension và thư viện Pagination.

Các phương thức cần biết

  • Tool::getThumbnail($filename) : lấy tên file Thumbnail bao gồm chuỗi _thumb
  • Tool::redirect($link) : chuyển trang
  • Tool::pagination($totalPage, $perPage, $currentPage, $range, $linkCallback): thư viện Pagination
  • Tool::convertToTimestamp($time): chuyển đổi thời gian san Timestamp trong MySQL
  • Tool::VNConvert($str): chuyển đổi ký tự Việt Nam sang ASCII

Lấy tên file Thumbnail

	
		echo Tool::getThumbnail('demo.jpg'); // demo_thumb.jpg

Chuyển trang

	
		Tool::redirect('http://example.com');

Thư viện Pagination

Thư viện này đòi hỏi bạn có kiến thức nhất định về Pagination như tính tổng số trang, lấy số trang hiện tại đang vào. Còn lại sẽ do thư viện sẽ tự động hết cho các bạn. Tuy nhiên đối số $linkCallback ít nhiều sẽ gây khó hiểu cho các bạn. Hãy thực hành qua ví dụ dưới đây nhé.

			
		Tool::pagination($totalPage, $perPage, $page, $range, function($currentPage) {
			// Đối số mặc định $currentPage sẽ được trả về trong vòng lặp ẩn của thư viện Pagination
			// Hàm này giúp các bạn viết lại đường dẫn cho link trả về của Pagination
		    return sprintf('#/admin/category/page/%s', $currentPage);
		});

Chuyển đổi thời gian sang Timestamp

	
		echo Tool::convertToTimestamp(time()); // 2015-08-01 10:20:30

Chuyển đổi ký tự Việt Nam sang ASCII

Thư viện này hỗ trợ các bạn trong việc Rewrite URL của tiêu đề bài viết hay bất cứ chuỗi nào đó mà bạn không muốn nó có khoảng trống như tên hình ảnh, tên file.

	
		echo Tool::VNConvert('Đây là chuỗi mẫu'); // day-la-chuoi-mau
