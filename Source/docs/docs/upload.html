Upload

Thư viện Upload giúp bạn có thể đơn giản hóa công việc Upload lên Server. Tuy nhiên, thư viện này được phát triển theo dạng "mì ăn liền" nên vì vậy không có những chức năng tinh chỉnh dành riêng cho lập trình viên. Lập trình viên có thể tự mình tạo ra thư viện Upload khác tốt hơn.

Các phương thức cần biết

  • Upload::setMimes($mimesArray) : thiết lập các chuẩn file được cho phép Upload
  • Upload::verify($file) : kiểm duyệt file Upload
  • Upload::put($file, $destination): đặt file Upload vào thư mục chứa file

Hướng dẫn chung

	
		$files = $_FILES;
		foreach ($files as $file) {
		    $verify = Upload::verify($file);
		    if ($verify !== true) {
		        switch ($verify) {
		            case 'maxsize':
		                $data['errors'][] = 'Dung lượng quá mức cho phép';
		                break;
		            case 'mimes':
		                $data['errors'][] = 'Định dạng không cho phép';
		                break;
		        }
		    }
		    if (!$data['errors']) {
		        $url = Upload::put($file, '../uploads');
		        echo $url;
		    }
		}
